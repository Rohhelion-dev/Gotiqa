# Gotiqa Frontend

Static frontend pages served by the Flask backend.

## Structure

```text
frontend/
  index.html
  home.html
  about.html
  products.html
  blog.html
  contact.html
  dashboard.html
  assets/
    css/
      about.css
      blog.css
      contact.css
      dashboard.css
      home.css
      index.css
      products.css
    js/
      contact.js
      dashboard.js
      index.js
      products.js
```

## Routes

Run the backend, then open:

- `http://localhost:5000/`
- `http://localhost:5000/home`
- `http://localhost:5000/about`
- `http://localhost:5000/products`
- `http://localhost:5000/blog`
- `http://localhost:5000/contact`
- `http://localhost:5000/dashboard`

CSS and JavaScript are served from `/assets/...`.
